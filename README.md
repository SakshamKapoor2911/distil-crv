# Distil-CRV: Generalizable Framework for Cost-Efficient CoT Verification

An industry-standard research framework for reasoning verification. It prioritizes:
- **Modularity:** Baseline (CRV) is a submodule; core contribution (Distil-CRV) is decoupled
- **Reproducibility:** Config-driven experiments, seed control, structured logging
- **Scalability:** Easy to swap models, datasets, auxiliary architectures

## Status
Currently in Phase 1: Foundation & Profiling.
